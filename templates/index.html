This portfolio showcases our group's laboratory exercises for the Data Structures and Algorithms course. Here, you can explore our implementations of data structures such as Linked Lists, Stacks, and Queues.

Each exercise demonstrates a specific concept:

  • Linked List: Includes operations such as adding, removing, and displaying movies in a watchlist.
  • Stack Implementation: The Shunting Yard algorithm for converting infix to postfix notation.
  • Queue Implementation: Standard Queue and De-Queue operations.

All features are integrated with Flask web application, allowing you to interact with the data structures in real-time.
